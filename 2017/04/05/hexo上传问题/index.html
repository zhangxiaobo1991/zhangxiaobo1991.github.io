hexo上传问题

经常上传的时候会遇到这样的问题

解决方案：
1、执行npm install hexo-deployer-git –save
2、配置Deployment首先，你需要为自己配置身份信息，打开命令行，然后输入：

git config –global user.name “yourname”
git config –global user.email “youremail”

3、删掉根目录下的文件夹：.deploy_git，重新deploy就可以了。
5、config.xml文件配置中，deploy中的repo使用git，而不是使用https

4、使用git bush工具，不用windows下的cmd工具(我试过)

参考链接： http://www.cnblogs.com/tenderwx/p/5783432.html
